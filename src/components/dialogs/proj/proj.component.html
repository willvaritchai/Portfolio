Modride: Ride Sharing in KMUTT is a ride-sharing web application for KMUTT users to find travel companions to similar destinations. It allows users to share trips, split costs, reduce traffic, and foster connections between different faculties. This web application was developed using the Angular front-end framework and the Spring Boot back-end framework.

Core Features

Authentication: Authenticated by Microsoft Identity Platform with KMUTT’s email domain only.
Driver Register: Users are able to register in order to become drivers by uploading the required information.
Trip Event: Users are able to view, create, edit, or delete their own event and make a request to join a trip event.
Group Chat: Users are able to communicate via group chat after a trip event owner has already accepted a request.

Technology Stack

Angular: front-end framework used for managing front-end display, featuring functionalities that meet the system requirements.
Spring Boot: A framework that supports Java and RESTful API development for Web Applications and Web Services.
MySQL: Database Management Software (DBMS) that can be operated via Shell as a Command line and through a Client App on the OS.
Figma/FigJam: Software for designing and organizing wireframes according to business requirements.
Docker: A software platform used to create, test, and manage containers within the infrastructure system.
Nginx: A Web Server for the system to display websites and manage Reverse Proxy to enhance the efficiency of communication between containers.
Postman: A tool used for testing API services developed by the Back End to ensure they meet the system's requirements.
Jira: Tools used for Project Management, such as defining requirements and tracking the status of tasks.
GitHub: A website providing Git Version Control for managing code versions, facilitating development from multiple developers, and storing the system's source code.

OASIP is an online appointment website where students can schedule or modify online meeting appointments with their course lecturers. This web application was developed using the Vue front-end framework and the Spring Boot back-end framework.

Core Features

Get All Event: Displaying the list of events where students have scheduled meetings with lecturers for specific online classes.
Booking Event: Creating events for scheduling meetings with lecturers for specific classes.
Event Detail: Checking the detailed information of the scheduled meeting events.
Edit Event: Editing the detailed information of the scheduled meeting events.
Cancel Event: Canceling scheduled meeting events.

FUNBOOK online book store is a website that offers a wide range of books for sale online, including educational books and books on astrology. This web application was developed using the Vue framework.

Description:
  • All books on the website are categorized to facilitate easy searching.
  • Book listings are displayed in card format and include details about each book, such as the title, author, etc.
  • Features allow users to add desired books to a wishlist, enabling them to purchase the books at a later date.

Core Features

Search for Books: Users can search for books by category using the search bar, specifying categories such as IT, AI, MySQL, coding, etc.
Wishlist: Users can add their favorite books to their wishlist by clicking the heart icon on the respective book. The book will then be saved to the wishlist page, which includes details of each saved book.
Remove from Wishlist: On the wishlist page, if users no longer want a book, they can remove it from the wishlist by clicking the Remove button.

WishYourGrade helps students assess and plan their academic performance, addressing their expectations about grades. This web application was developed using the Vue framework.

Description:
  • Upon entering, users will see a form where they can input information.
  • Users can enter the names of subjects, grade levels, and course credits.
  • Below the input section, there is information about GPA, explaining its meaning and the calculation method.
  • Different GPA levels are displayed uniquely; for instance, if the GPA is above 3.50, the background color will be green.
  • GPA calculation is in real-time, meaning users can adjust the grade levels or credits, and the GPA result will update accordingly.

Core Features

Add Rows: Users can add more rows to accommodate the number of subjects they wish to enter by clicking the blue button with a plus icon.
Reset: Users can reset all the entered information by clicking the gray button with a cross icon.
